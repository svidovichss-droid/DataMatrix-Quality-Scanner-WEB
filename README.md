# Data Matrix Quality Scanner

Сканер качества печати Data Matrix согласно **ГОСТ Р 57302-2016** (ISO/IEC 15415).

## Возможности

- ✅ Проверка качества печати по 7 параметрам ГОСТ
- ✅ Работа с промышленными камерами в реальном времени
- ✅ Поддержка конвейерного режима с внешним триггером
- ✅ Автоматическое сохранение результатов и изображений
- ✅ Экспорт отчетов в CSV/JSON
- ✅ Статистика и аналитика качества

## Параметры проверки (ГОСТ Р 57302-2016)

| Параметр | Описание | Минимум (C) |
|----------|----------|-------------|
| Decode | Декодирование | Обязательно |
| Symbol Contrast | Контраст символа | 40% |
| Min Reflectance | Мин. отражаемость | ≤ 50% SC |
| Min Edge Contrast | Мин. контраст края | 15% |
| Modulation | Модуляция | 0.50 |
| Defects | Дефекты | ≤ 1.5% |
| Decodability | Декодируемость | 40% |
