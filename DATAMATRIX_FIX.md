# Исправление проблемы с захватом Data Matrix камерой

## Проблема
Камера не распознаёт и не захватывает коды Data Matrix.

## Решение

### 1. Улучшена предобработка изображений

Добавлен новый метод `_preprocess_aggressive()` в `scanner/datamatrix_decoder.py`:
- Использует CLAHE (Contrast Limited Adaptive Histogram Equalization) для улучшения локального контраста
- Применяет более агрессивные морфологические операции
- Использует бинаризацию Оцу для оптимального порога

### 2. Расширен набор вариантов обработки

Все методы декодирования теперь используют 5 вариантов изображения вместо 3:
1. Оригинальное изображение
2. Стандартная предобработка (`_preprocess`)
3. Агрессивная предобработка (`_preprocess_aggressive`) - **НОВОЕ**
4. Инверсия стандартной предобработки
5. Инверсия агрессивной предобработки - **НОВОЕ**

### 3. Обновленные методы

Изменения применены ко всем ключевым методам:
- `_detect_with_pylibdmtx()` - поиск кодов
- `_decode_region_with_pylibdmtx()` - декодирование области
- `_decode_with_pylibdmtx()` - полное декодирование
- `_decode_region_with_pyzbar()` - декодирование через pyzbar
- `_decode_with_pyzbar()` - полное декодирование через pyzbar

### 4. Технические детали

#### Стандартная предобработка (`_preprocess`):
```python
- GaussianBlur (3x3) - удаление шума
- Sharpening kernel - улучшение границ
- Adaptive Threshold (Gaussian) - бинаризация
- Morphology (dilate/erode) - очистка
```

#### Агрессивная предобработка (`_preprocess_aggressive`):
```python
- GaussianBlur (5x5) - сильное удаление шума
- CLAHE (clipLimit=3.0, tileGridSize=8x8) - улучшение контраста
- Otsu thresholding - оптимальная бинаризация
- Morphology (dilate x2, erode x1) - агрессивная очистка
```

## Преимущества

1. **Лучшая работа при плохом освещении** - CLAHE улучшает видимость кода
2. **Устойчивость к бликам** - инверсия помогает при засветке
3. **Работа с низкоконтрастными кодами** - агрессивная обработка выявляет скрытые детали
4. **Поддержка повреждённых кодов** - множественные попытки увеличивают шанс успеха

## Примечания

- Время обработки может немного увеличиться из-за дополнительных попыток

- Методы работают каскадно: если первый способ не сработал, пробуется следующий
